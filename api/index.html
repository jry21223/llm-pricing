🔌 LLM Pricing API

RESTful API 接口，查询各大模型价格

📍 基本信息

Base URL: https://jry21223.github.io/llm-pricing

也支持 Vercel 部署：https://llm-pricing.vercel.app

所有接口返回 JSON 格式，支持 CORS。

⚡ Vercel 动态 API（推荐）

部署到 Vercel 后可使用动态查询，无需重新生成静态文件。

部署方式：vercel --prod

GET /api/models

查看所有模型，支持筛选：

参数示例说明
?provider=deepseek按厂商筛选
?search=gpt搜索模型名或厂商名
?minPrice=0.1最低 Input 价格
?maxPrice=2最高 Input 价格
?currency=CNY币种过滤
?sort=input排序 (input/output/context)

GET /api/models/:slug

特定厂商：/api/models/deepseek

GET /api/compare?models=a,b,c

对比模型：/api/compare?models=gpt-5.5,claude-opus,qwen3.7

GET /api/stats

统计数据（同静态版）

GET /api/providers

厂商列表

📦 调用示例

# 获取所有模型
curl https://jry21223.github.io/llm-pricing/api/v1/models.json

# 获取 OpenAI 模型（Vercel 动态 API）
curl /api/models?provider=openai

# 搜索 GPT 相关模型
curl /api/models?search=gpt&sort=input

# 对比三款热门模型
curl /api/compare?models=gpt-5.5,claude-opus-4.7,qwen3.7-max

# 用 JavaScript fetch
fetch('/api/models?minPrice=0&maxPrice=1¤cy=USD')
  .then(r => r.json())
  .then(data => console.log(data))
